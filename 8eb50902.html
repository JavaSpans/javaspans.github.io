з¬¬дёҖз«  Objectзұ»

1.1 жҰӮиҝ°

java.lang.Objectзұ»жҳҜJavaиҜӯиЁҖдёӯзҡ„ж №зұ»пјҢеҚіжүҖжңүзұ»зҡ„зҲ¶зұ»гҖӮе®ғдёӯжҸҸиҝ°зҡ„жүҖжңүж–№жі•еӯҗзұ»йғҪеҸҜд»ҘдҪҝз”ЁгҖӮеңЁеҜ№иұЎе®һдҫӢеҢ–зҡ„ж—¶еҖҷпјҢжңҖз»Ҳжүҫзҡ„зҲ¶зұ»е°ұжҳҜObjectгҖӮ

еҰӮжһңдёҖдёӘзұ»жІЎжңүзү№еҲ«жҢҮе®ҡзҲ¶зұ»пјҢ йӮЈд№Ҳй»ҳи®ӨеҲҷз»§жүҝиҮӘObjectзұ»гҖӮдҫӢеҰӮпјҡ

ж №жҚ®JDKжәҗд»Јз ҒеҸҠObjectзұ»зҡ„APIж–ҮжЎЈпјҢObjectзұ»еҪ“дёӯеҢ…еҗ«зҡ„ж–№жі•жңү11дёӘгҖӮд»ҠеӨ©жҲ‘д»¬дё»иҰҒеӯҰд№ е…¶дёӯзҡ„2дёӘпјҡ

  • public String toString()пјҡиҝ”еӣһиҜҘеҜ№иұЎзҡ„еӯ—з¬ҰдёІиЎЁзӨәгҖӮ
  • public boolean equals(Object obj)пјҡжҢҮзӨәе…¶д»–жҹҗдёӘеҜ№иұЎжҳҜеҗҰдёҺжӯӨеҜ№иұЎвҖңзӣёзӯүвҖқгҖӮ

1.2 toStringж–№жі•

ж–№жі•ж‘ҳиҰҒ

  • public String toString()пјҡиҝ”еӣһиҜҘеҜ№иұЎзҡ„еӯ—з¬ҰдёІиЎЁзӨәгҖӮ

toStringж–№жі•иҝ”еӣһиҜҘеҜ№иұЎзҡ„еӯ—з¬ҰдёІиЎЁзӨәпјҢе…¶е®һиҜҘеӯ—з¬ҰдёІеҶ…е®№е°ұжҳҜеҜ№иұЎзҡ„зұ»еһӢ+@+еҶ…еӯҳең°еқҖеҖјгҖӮ

з”ұдәҺtoStringж–№жі•иҝ”еӣһзҡ„з»“жһңжҳҜеҶ…еӯҳең°еқҖпјҢиҖҢеңЁејҖеҸ‘дёӯпјҢз»ҸеёёйңҖиҰҒжҢүз…§еҜ№иұЎзҡ„еұһжҖ§еҫ—еҲ°зӣёеә”зҡ„еӯ—з¬ҰдёІиЎЁзҺ°еҪўејҸпјҢеӣ жӯӨд№ҹйңҖиҰҒйҮҚеҶҷе®ғгҖӮ

иҰҶзӣ–йҮҚеҶҷ

еҰӮжһңдёҚеёҢжңӣдҪҝз”ЁtoStringж–№жі•зҡ„й»ҳи®ӨиЎҢдёәпјҢеҲҷеҸҜд»ҘеҜ№е®ғиҝӣиЎҢиҰҶзӣ–йҮҚеҶҷгҖӮдҫӢеҰӮиҮӘе®ҡд№үзҡ„Personзұ»пјҡ

еңЁIntelliJ IDEAдёӯпјҢеҸҜд»ҘзӮ№еҮ»CodeиҸңеҚ•дёӯзҡ„Generate...пјҢд№ҹеҸҜд»ҘдҪҝз”Ёеҝ«жҚ·й”®alt+insertпјҢзӮ№еҮ»toString()йҖүйЎ№гҖӮйҖүжӢ©йңҖиҰҒеҢ…еҗ«зҡ„жҲҗе‘ҳеҸҳйҮҸе№¶зЎ®е®ҡгҖӮеҰӮдёӢеӣҫжүҖзӨәпјҡ

е°ҸиҙҙеЈ«пјҡ еңЁжҲ‘д»¬зӣҙжҺҘдҪҝз”Ёиҫ“еҮәиҜӯеҸҘиҫ“еҮәеҜ№иұЎеҗҚзҡ„ж—¶еҖҷ,е…¶е®һйҖҡиҝҮиҜҘеҜ№иұЎи°ғз”ЁдәҶе…¶toString()ж–№жі•гҖӮ

1.3 equalsж–№жі•

ж–№жі•ж‘ҳиҰҒ

  • public boolean equals(Object obj)пјҡжҢҮзӨәе…¶д»–жҹҗдёӘеҜ№иұЎжҳҜеҗҰдёҺжӯӨеҜ№иұЎвҖңзӣёзӯүвҖқгҖӮ

и°ғз”ЁжҲҗе‘ҳж–№жі•equalsе№¶жҢҮе®ҡеҸӮж•°дёәеҸҰдёҖдёӘеҜ№иұЎпјҢеҲҷеҸҜд»ҘеҲӨж–ӯиҝҷдёӨдёӘеҜ№иұЎжҳҜеҗҰжҳҜзӣёеҗҢзҡ„гҖӮиҝҷйҮҢзҡ„вҖңзӣёеҗҢвҖқжңүй»ҳи®Өе’ҢиҮӘе®ҡд№үдёӨз§Қж–№ејҸгҖӮ

й»ҳи®Өең°еқҖжҜ”иҫғ

еҰӮжһңжІЎжңүиҰҶзӣ–йҮҚеҶҷequalsж–№жі•пјҢйӮЈд№ҲObjectзұ»дёӯй»ҳи®ӨиҝӣиЎҢ==иҝҗз®—з¬Ұзҡ„еҜ№иұЎең°еқҖжҜ”иҫғпјҢеҸӘиҰҒдёҚжҳҜеҗҢдёҖдёӘеҜ№иұЎпјҢз»“жһңеҝ…з„¶дёәfalseгҖӮ

еҜ№иұЎеҶ…е®№жҜ”иҫғ

еҰӮжһңеёҢжңӣиҝӣиЎҢеҜ№иұЎзҡ„еҶ…е®№жҜ”иҫғпјҢеҚіжүҖжңүжҲ–жҢҮе®ҡзҡ„йғЁеҲҶжҲҗе‘ҳеҸҳйҮҸзӣёеҗҢе°ұеҲӨе®ҡдёӨдёӘеҜ№иұЎзӣёеҗҢпјҢеҲҷеҸҜд»ҘиҰҶзӣ–йҮҚеҶҷequalsж–№жі•гҖӮдҫӢеҰӮпјҡ

иҝҷж®өд»Јз Ғе……еҲҶиҖғиҷ‘дәҶеҜ№иұЎдёәз©әгҖҒзұ»еһӢдёҖиҮҙзӯүй—®йўҳпјҢдҪҶж–№жі•еҶ…е®№е№¶дёҚе”ҜдёҖгҖӮеӨ§еӨҡж•°IDEйғҪеҸҜд»ҘиҮӘеҠЁз”ҹжҲҗequalsж–№жі•зҡ„д»Јз ҒеҶ…е®№гҖӮеңЁIntelliJ IDEAдёӯпјҢеҸҜд»ҘдҪҝз”ЁCodeиҸңеҚ•дёӯзҡ„GenerateвҖҰйҖүйЎ№пјҢд№ҹеҸҜд»ҘдҪҝз”Ёеҝ«жҚ·й”®alt+insertпјҢе№¶йҖүжӢ©equals() and hashCode()иҝӣиЎҢиҮӘеҠЁд»Јз Ғз”ҹжҲҗгҖӮеҰӮдёӢеӣҫжүҖзӨәпјҡ

tipsпјҡObjectзұ»еҪ“дёӯзҡ„hashCodeзӯүе…¶д»–ж–№жі•пјҢд»ҠеҗҺеӯҰд№ гҖӮ

1.4 Objectsзұ»

еңЁеҲҡжүҚIDEAиҮӘеҠЁйҮҚеҶҷequalsд»Јз ҒдёӯпјҢдҪҝз”ЁеҲ°дәҶjava.util.Objectsзұ»пјҢйӮЈд№ҲиҝҷдёӘзұ»жҳҜд»Җд№Ҳе‘ўпјҹ

еңЁJDK7ж·»еҠ дәҶдёҖдёӘObjectsе·Ҙе…·зұ»пјҢе®ғжҸҗдҫӣдәҶдёҖдәӣж–№жі•жқҘж“ҚдҪңеҜ№иұЎпјҢе®ғз”ұдёҖдәӣйқҷжҖҒзҡ„е®һз”Ёж–№жі•з»„жҲҗпјҢиҝҷдәӣж–№жі•жҳҜnull-saveпјҲз©әжҢҮй’Ҳе®үе…Ёзҡ„пјүжҲ–null-tolerantпјҲе®№еҝҚз©әжҢҮй’Ҳзҡ„пјүпјҢз”ЁдәҺи®Ўз®—еҜ№иұЎзҡ„hashcodeгҖҒиҝ”еӣһеҜ№иұЎзҡ„еӯ—з¬ҰдёІиЎЁзӨәеҪўејҸгҖҒжҜ”иҫғдёӨдёӘеҜ№иұЎгҖӮ

еңЁжҜ”иҫғдёӨдёӘеҜ№иұЎзҡ„ж—¶еҖҷпјҢObjectзҡ„equalsж–№жі•е®№жҳ“жҠӣеҮәз©әжҢҮй’ҲејӮеёёпјҢиҖҢObjectsзұ»дёӯзҡ„equalsж–№жі•е°ұдјҳеҢ–дәҶиҝҷдёӘй—®йўҳгҖӮж–№жі•еҰӮдёӢпјҡ

  • public static boolean equals(Object a, Object b):еҲӨж–ӯдёӨдёӘеҜ№иұЎжҳҜеҗҰзӣёзӯүгҖӮ

жҲ‘д»¬еҸҜд»ҘжҹҘзңӢдёҖдёӢжәҗз ҒпјҢеӯҰд№ дёҖдёӢпјҡ

2.1 Dateзұ»

жҰӮиҝ°

java.util.Dateзұ» иЎЁзӨәзү№е®ҡзҡ„зһ¬й—ҙпјҢзІҫзЎ®еҲ°жҜ«з§’гҖӮ

з»§з»ӯжҹҘйҳ…Dateзұ»зҡ„жҸҸиҝ°пјҢеҸ‘зҺ°DateжӢҘжңүеӨҡдёӘжһ„йҖ еҮҪж•°пјҢеҸӘжҳҜйғЁеҲҶе·Із»ҸиҝҮж—¶пјҢдҪҶжҳҜе…¶дёӯжңүжңӘиҝҮж—¶зҡ„жһ„йҖ еҮҪж•°еҸҜд»ҘжҠҠжҜ«з§’еҖјиҪ¬жҲҗж—ҘжңҹеҜ№иұЎгҖӮ

  • public Date()пјҡеҲҶй…ҚDateеҜ№иұЎе№¶еҲқе§ӢеҢ–жӯӨеҜ№иұЎпјҢд»ҘиЎЁзӨәеҲҶй…Қе®ғзҡ„ж—¶й—ҙпјҲзІҫзЎ®еҲ°жҜ«з§’пјүгҖӮ
  • public Date(long date)пјҡеҲҶй…ҚDateеҜ№иұЎе№¶еҲқе§ӢеҢ–жӯӨеҜ№иұЎпјҢд»ҘиЎЁзӨәиҮӘд»Һж ҮеҮҶеҹәеҮҶж—¶й—ҙпјҲз§°дёәвҖңеҺҶе…ғпјҲepochпјүвҖқпјҢеҚі1970е№ҙ1жңҲ1ж—Ҙ00:00:00 GMTпјүд»ҘжқҘзҡ„жҢҮе®ҡжҜ«з§’ж•°гҖӮ

tips: з”ұдәҺжҲ‘д»¬еӨ„дәҺдёңе…«еҢәпјҢжүҖд»ҘжҲ‘д»¬зҡ„еҹәеҮҶж—¶й—ҙдёә1970е№ҙ1жңҲ1ж—Ҙ8ж—¶0еҲҶ0з§’гҖӮ

з®ҖеҚ•жқҘиҜҙпјҡдҪҝз”Ёж— еҸӮжһ„йҖ пјҢеҸҜд»ҘиҮӘеҠЁи®ҫзҪ®еҪ“еүҚзі»з»ҹж—¶й—ҙзҡ„жҜ«з§’ж—¶еҲ»пјӣжҢҮе®ҡlongзұ»еһӢзҡ„жһ„йҖ еҸӮж•°пјҢеҸҜд»ҘиҮӘе®ҡд№үжҜ«з§’ж—¶еҲ»гҖӮдҫӢеҰӮпјҡ

tips:еңЁдҪҝз”Ёprintlnж–№жі•ж—¶пјҢдјҡиҮӘеҠЁи°ғз”ЁDateзұ»дёӯзҡ„toStringж–№жі•гҖӮDateзұ»еҜ№Objectзұ»дёӯзҡ„toStringж–№жі•иҝӣиЎҢдәҶиҰҶзӣ–йҮҚеҶҷпјҢжүҖд»Ҙз»“жһңдёәжҢҮе®ҡж јејҸзҡ„еӯ—з¬ҰдёІгҖӮ

  • ж јејҸеҢ–пјҡжҢүз…§жҢҮе®ҡзҡ„ж јејҸпјҢд»ҺDateеҜ№иұЎиҪ¬жҚўдёәStringеҜ№иұЎгҖӮ

2.4 Calendarзұ»

жҰӮеҝө

ж—ҘеҺҶжҲ‘д»¬йғҪи§ҒиҝҮ

java.util.CalendarжҳҜж—ҘеҺҶзұ»пјҢеңЁDateеҗҺеҮәзҺ°пјҢжӣҝжҚўжҺүдәҶи®ёеӨҡDateзҡ„ж–№жі•гҖӮиҜҘзұ»е°ҶжүҖжңүеҸҜиғҪз”ЁеҲ°зҡ„ж—¶й—ҙдҝЎжҒҜе°ҒиЈ…дёәйқҷжҖҒжҲҗе‘ҳеҸҳйҮҸпјҢж–№дҫҝиҺ·еҸ–гҖӮж—ҘеҺҶзұ»е°ұжҳҜж–№дҫҝиҺ·еҸ–еҗ„дёӘж—¶й—ҙеұһжҖ§зҡ„гҖӮ

иҺ·еҸ–ж–№ејҸ

CalendarдёәжҠҪиұЎзұ»пјҢз”ұдәҺиҜӯиЁҖж•Ҹж„ҹжҖ§пјҢCalendarзұ»еңЁеҲӣе»әеҜ№иұЎж—¶е№¶йқһзӣҙжҺҘеҲӣе»әпјҢиҖҢжҳҜйҖҡиҝҮйқҷжҖҒж–№жі•еҲӣе»әпјҢиҝ”еӣһеӯҗзұ»еҜ№иұЎпјҢеҰӮдёӢпјҡ

CalendarйқҷжҖҒж–№жі•

  • public static Calendar getInstance()пјҡдҪҝз”Ёй»ҳи®Өж—¶еҢәе’ҢиҜӯиЁҖзҺҜеўғиҺ·еҫ—дёҖдёӘж—ҘеҺҶ

дҫӢеҰӮпјҡ

ж №жҚ®Calendarзұ»зҡ„APIж–ҮжЎЈпјҢеёёз”Ёж–№жі•жңүпјҡ

  • public int get(int field)пјҡиҝ”еӣһз»ҷе®ҡж—ҘеҺҶеӯ—ж®өзҡ„еҖјгҖӮ

з¬¬дёүз«  Systemзұ»

java.lang.Systemзұ»дёӯжҸҗдҫӣдәҶеӨ§йҮҸзҡ„йқҷжҖҒж–№жі•пјҢеҸҜд»ҘиҺ·еҸ–дёҺзі»з»ҹзӣёе…ізҡ„дҝЎжҒҜжҲ–зі»з»ҹзә§ж“ҚдҪңпјҢеңЁSystemзұ»зҡ„APIж–ҮжЎЈдёӯпјҢеёёз”Ёзҡ„ж–№жі•жңүпјҡ

  • public static long currentTimeMillis()пјҡиҝ”еӣһд»ҘжҜ«з§’дёәеҚ•дҪҚзҡ„еҪ“еүҚж—¶й—ҙгҖӮ
  • public static void arraycopy(Object src, int srcPos, Object dest, int destPos, int length)пјҡе°Ҷж•°з»„дёӯжҢҮе®ҡзҡ„ж•°жҚ®жӢ·иҙқеҲ°еҸҰдёҖдёӘж•°з»„дёӯгҖӮ

3.1 currentTimeMillisж–№жі•

е®һйҷ…дёҠпјҢcurrentTimeMillisж–№жі•е°ұжҳҜ иҺ·еҸ–еҪ“еүҚзі»з»ҹж—¶й—ҙдёҺ1970е№ҙ01жңҲ01ж—Ҙ00:00зӮ№д№Ӣй—ҙзҡ„жҜ«з§’е·®еҖј

  • public static void arraycopy(Object src, int srcPos, Object dest, int destPos, int length)пјҡе°Ҷж•°з»„дёӯжҢҮе®ҡзҡ„ж•°жҚ®жӢ·иҙқеҲ°еҸҰдёҖдёӘж•°з»„дёӯгҖӮ

4.1 еӯ—з¬ҰдёІжӢјжҺҘй—®йўҳ

з”ұдәҺStringзұ»зҡ„еҜ№иұЎеҶ…е®№дёҚеҸҜж”№еҸҳпјҢжүҖд»ҘжҜҸеҪ“иҝӣиЎҢеӯ—з¬ҰдёІжӢјжҺҘж—¶пјҢжҖ»жҳҜдјҡеңЁеҶ…еӯҳдёӯеҲӣе»әдёҖдёӘж–°зҡ„еҜ№иұЎгҖӮдҫӢеҰӮпјҡ

еңЁAPIдёӯеҜ№Stringзұ»жңүиҝҷж ·зҡ„жҸҸиҝ°пјҡеӯ—з¬ҰдёІжҳҜеёёйҮҸпјҢе®ғд»¬зҡ„еҖјеңЁеҲӣе»әеҗҺдёҚиғҪиў«жӣҙж”№гҖӮ

ж №жҚ®иҝҷеҸҘиҜқеҲҶжһҗжҲ‘д»¬зҡ„д»Јз ҒпјҢе…¶е®һжҖ»е…ұдә§з”ҹдәҶдёүдёӘеӯ—з¬ҰдёІпјҢеҚі"Hello"гҖҒ"World"е’Ң"HelloWorld"гҖӮеј•з”ЁеҸҳйҮҸsйҰ–е…ҲжҢҮеҗ‘HelloеҜ№иұЎпјҢжңҖз»ҲжҢҮеҗ‘жӢјжҺҘеҮәжқҘзҡ„ж–°еӯ—з¬ҰдёІеҜ№иұЎпјҢеҚіHelloWord гҖӮ

з”ұжӯӨеҸҜзҹҘпјҢеҰӮжһңеҜ№еӯ—з¬ҰдёІиҝӣиЎҢжӢјжҺҘж“ҚдҪңпјҢжҜҸж¬ЎжӢјжҺҘпјҢйғҪдјҡжһ„е»әдёҖдёӘж–°зҡ„StringеҜ№иұЎпјҢж—ўиҖ—ж—¶пјҢеҸҲжөӘиҙ№з©әй—ҙгҖӮдёәдәҶи§ЈеҶіиҝҷдёҖй—®йўҳпјҢеҸҜд»ҘдҪҝз”Ёjava.lang.StringBuilderзұ»гҖӮ

4.2 StringBuilderжҰӮиҝ°

жҹҘйҳ…java.lang.StringBuilderзҡ„APIпјҢStringBuilderеҸҲз§°дёәеҸҜеҸҳеӯ—з¬ҰеәҸеҲ—пјҢе®ғжҳҜдёҖдёӘзұ»дјјдәҺ String зҡ„еӯ—з¬ҰдёІзј“еҶІеҢәпјҢйҖҡиҝҮжҹҗдәӣж–№жі•и°ғз”ЁеҸҜд»Ҙж”№еҸҳиҜҘеәҸеҲ—зҡ„й•ҝеәҰе’ҢеҶ…е®№гҖӮ

еҺҹжқҘStringBuilderжҳҜдёӘеӯ—з¬ҰдёІзҡ„зј“еҶІеҢәпјҢеҚіе®ғжҳҜдёҖдёӘе®№еҷЁпјҢе®№еҷЁдёӯеҸҜд»ҘиЈ…еҫҲеӨҡеӯ—з¬ҰдёІгҖӮе№¶дё”иғҪеӨҹеҜ№е…¶дёӯзҡ„еӯ—з¬ҰдёІиҝӣиЎҢеҗ„з§Қж“ҚдҪңгҖӮ

е®ғзҡ„еҶ…йғЁжӢҘжңүдёҖдёӘж•°з»„з”ЁжқҘеӯҳж”ҫеӯ—з¬ҰдёІеҶ…е®№пјҢиҝӣиЎҢеӯ—з¬ҰдёІжӢјжҺҘж—¶пјҢзӣҙжҺҘеңЁж•°з»„дёӯеҠ е…Ҙж–°еҶ…е®№гҖӮStringBuilderдјҡиҮӘеҠЁз»ҙжҠӨж•°з»„зҡ„жү©е®№гҖӮеҺҹзҗҶеҰӮдёӢеӣҫжүҖзӨәпјҡ(й»ҳи®Ө16еӯ—з¬Ұз©әй—ҙпјҢи¶…иҝҮиҮӘеҠЁжү©е……)
